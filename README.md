# Rocketseat - Curso JavaScript ES6

## Módulos: 
- Conceitos
- Webpack Server
- Async/ await
- Aplicação com ES6+

## Necessário ter instalado:
- Node.js + NPM
- Yarn

## Executando:

- Pasta Aplicação ou Desafios: 

```javascript
    npm install
    yarn dev // inicia na porta 8080 - http://localhost:8080/
```
